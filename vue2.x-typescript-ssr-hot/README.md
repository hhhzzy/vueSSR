# vue2.x + typescript + ssr
```
配置开发环境的热更新，引入element ui ，配置正式环境的部署
```

### Project setup
```
npm install
```

### open http://localhost:8088
```
npm run server
```

### build for production
```
npm run build
```

### serve in production
```
npm run start
```

### Lints and fixes files
```
npm run lint
```
