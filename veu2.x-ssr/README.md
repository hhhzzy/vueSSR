# veu2.x + typescript + ssr

### Project setup
```
npm install
```

### build for client
```
npm run build
```

### build for server
```
npm run build:server
```

### open http://localhost:8088
```
npm run server
```

### Lints and fixes files
```
npm run lint
```

### tips
```
每次更新代码必须重新打包，没做热重载
```
